# 📦 代码打包与迁移指南

为了让其他人在不安装开发环境的情况下运行本项目，请按照以下步骤操作：

## 1. 准备前端静态文件 (在您的机器)
1. 打开终端，进入 `frontend` 目录。
2. 运行：`npm run build`。
3. 运行完成后，你会看到一个 `dist` 文件夹。
4. 将 `dist` 文件夹内的**所有内容**复制到 `backend/static` 目录下（如果没有 `static` 文件夹请手动新建一个）。

## 2. 整理压缩包 (发给对方的内容)
新建一个文件夹（如 `douyin_crawler_release`），将以下内容拷贝进去：

- `backend/` 文件夹 (⚠️ **重要：删除里面的 `venv` 和 `__pycache__` 文件夹**)
- `start.bat` 启动脚本
- `.env` 配置文件 (确保里面填好了 API 密钥)
- `README.md` (可选)

## 3. 对方的操作步骤
对方收到压缩包并解压后，只需执行以下两步：

1. **安装环境**：确保电脑安装了 [Python 3.9 或更高版本](https://www.python.org/)。
2. **一键启动**：双击运行 `start.bat`。
   - 脚本会自动创建虚拟环境、安装所有库并启动服务。
3. **访问界面**：在浏览器打开 `http://localhost:8000` 即可使用。

---

### ⚠️ 注意事项
- **端口冲突**：如果对方机器的 8000 端口被占用，请在 `.env` 或 `main.py` 中修改端口。
- **浏览器依赖**：本项目使用了 `DrissionPage`，启动时会自动调用系统安装的 Chrome 或 Edge 浏览器。请确保对方电脑上至少安装了其中之一。
- **数据迁移**：如果您想把采集到的视频也发给对方，请连同 `backend/downloads` 和 `backend/data.db`（SQLite 数据库文件）一起打包。如果只想发代码，就不要包含这两个文件。
